# 学生管理系统

基于FastAPI的现代化学生管理系统，支持Web界面操作，提供完整的班级、学生、成绩管理功能。

## 🌟 项目特点

- ✅ **现代化Web界面** - 美观的登录界面和管理后台
- ✅ **直观操作** - 所见即所得，鼠标点击即可完成所有操作
- ✅ **行内编辑** - 表格内直接编辑，无需弹窗，操作更便捷
- ✅ **用户认证** - 基于JWT的安全认证系统
- ✅ **权限管理** - 支持管理员、教师、学生三种角色
- ✅ **班级管理** - 添加、查看、编辑、删除班级信息
- ✅ **学生管理** - 添加、查看、编辑、删除学生信息
- ✅ **成绩管理** - 录入、查看、编辑、删除成绩记录
- ✅ **SQLite数据库** - 无需额外安装数据库，开箱即用
- ✅ **响应式设计** - 适配不同屏幕尺寸

## 🚀 快速开始

### 环境要求

- Python 3.8 或更高版本
- 现代浏览器（Chrome、Firefox、Edge等）

### 安装步骤

1. **安装依赖**
   ```bash
   cd student_management_system
   pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
   ```

2. **初始化数据库**（首次运行）
   ```bash
   python init_db.py
   ```

3. **启动服务**
   ```bash
   python main.py
   ```

4. **访问系统**
   
   打开浏览器访问：http://localhost:8000/

## 📖 使用指南

### 1. 登录系统

1. 访问 http://localhost:8000/
2. 输入用户名：`admin`
3. 输入密码：`admin123`
4. 点击"登录"按钮

### 2. 班级管理

**添加班级**
1. 点击左侧菜单"班级管理"
2. 输入班级名称（如：三年级1班）
3. 输入班主任姓名
4. 点击"添加班级"按钮

**编辑班级**
1. 在班级列表中找到要编辑的班级
2. 点击"编辑"按钮
3. 在表格中直接修改班级名称和班主任
4. 点击"保存"按钮完成修改

**删除班级**
- 在班级列表中点击"删除"按钮
- 注意：班级中有学生时无法删除

### 3. 学生管理

**添加学生**
1. 点击左侧菜单"学生管理"
2. 填写学生信息：
   - 学号（必填，添加后不可修改）
   - 姓名（必填）
   - 性别
   - 出生日期（添加后不可修改）
   - 班级（添加后不可修改）
   - 联系电话
3. 点击"添加学生"按钮

**编辑学生**
1. 在学生列表中找到要编辑的学生
2. 点击"编辑"按钮
3. 在表格中直接修改：姓名、性别、联系电话
4. 点击"保存"按钮完成修改

**删除学生**
- 在学生列表中点击"删除"按钮

### 4. 成绩管理

**成绩录入模式**
系统支持两种成绩录入模式：

- **单条录入（默认）**：适合零星输入成绩。
  - 学生选择升级为带搜索功能的下拉框，支持拼音、姓名、学号模糊搜索。
  - 输入课程名称、考试日期、分数后，点击【添加单条成绩】按钮。

- **批量录入（按班级）**：适合大量成绩录入。
  - 选择班级、课程名称、考试日期后点击【⚡ 生成成绩花名册】按钮。
  - 下方将出现该班学生列表及对应空白分数字段。
  - 支持键盘快速录入，Tab键自动跳转输入框。
  - 输入完成后，点击【一键提交全班成绩】按钮，一次性保存所有成绩。

**编辑成绩**
1. 在成绩列表中找到要编辑的成绩记录
2. 点击"编辑"按钮
3. 在表格中直接修改：课程名称、分数、考试日期
4. 点击"保存"按钮完成修改

**添加成绩**
1. 点击左侧菜单"成绩管理"
2. 选择学生或班级
3. 输入课程名称、考试日期、分数
4. 点击对应录入按钮

**删除成绩**
- 在成绩列表中点击"删除"按钮

## 🛠️ 技术栈

### 后端技术
- **FastAPI** - 现代高性能Web框架
- **SQLAlchemy** - ORM数据库工具
- **SQLite** - 轻量级数据库
- **JWT** - 安全认证机制
- **Pydantic** - 数据验证

### 前端技术
- **HTML5** - 现代化页面结构
- **CSS3** - 美观的样式设计
- **JavaScript** - 动态交互功能
- **Fetch API** - 异步数据交互
