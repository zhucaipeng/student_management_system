## 💻 本地开发部署

### 方案一：快速启动（推荐新手）

#### Windows系统

1. **下载项目**

## 🚀 生产环境部署

### 方案一：Gunicorn + Nginx（Linux推荐）

#### 1. 安装Gunicorn
```bash
pip install gunicorn
```

#### 2. 创建启动脚本
创建 `start.sh`:
```bash
#!/bin/bash
gunicorn -w 4 -k uvicorn.workers.UvicornWorker -b 0.0.0.0:8000 main:app
```

#### 3. 配置Nginx
创建 `/etc/nginx/sites-available/student_management`:
```nginx
server {
    listen 80;
    server_name your-domain.com;

    location / {
        proxy_pass http://127.0.0.1:8000;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
    }
}
```

#### 4. 启动服务
```bash
chmod +x start.sh
./start.sh &
```

### 方案二：Systemd服务（Linux系统服务）

#### 1. 创建服务文件
创建 `/etc/systemd/system/student-management.service`:
```ini
[Unit]
Description=Student Management System
After=network.target

[Service]
Type=simple
User=www-data
WorkingDirectory=/opt/student_management_system
Environment=PATH=/opt/student_management_system/venv/bin
ExecStart=/opt/student_management_system/venv/bin/gunicorn -w 4 -k uvicorn.workers.UvicornWorker -b 0.0.0.0:8000 main:app
Restart=always

[Install]
WantedBy=multi-user.target
```

#### 2. 启动服务
```bash
sudo systemctl daemon-reload
sudo systemctl start student-management
sudo systemctl enable student-management
```

### 方案三：Docker部署

#### 1. 创建Dockerfile
创建 `Dockerfile`:
```dockerfile
FROM python:3.11-slim

WORKDIR /app

COPY requirements.txt .
RUN pip install --no-cache-dir -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple

COPY . .

EXPOSE 8000

CMD ["python", "main.py"]
```

#### 2. 构建镜像
```bash
docker build -t student-management .
```

#### 3. 运行容器
```bash
docker run -d -p 8000:8000 -v $(pwd)/student_management.db:/app/student_management.db student-management
```

### 方案四：Windows服务部署

#### 1. 安装NSSM
下载地址：https://nssm.cc/download

#### 2. 创建Windows服务
```bash
nssm install StudentManagement "C:\Python311\python.exe" "C:\student_management_system\main.py"

```

## ❓ 常见问题

### 1. 端口被占用

**错误信息**: `Address already in use`

**解决方案**:
```bash
# 查找占用进程
netstat -ano | findstr :8000  # Windows
lsof -i :8000                  # Linux/Mac

# 结束进程或更改端口
# 修改 main.py 最后一行
uvicorn.run(app, host="0.0.0.0", port=8001)
```

### 2. 数据库权限问题

**错误信息**: `Permission denied: student_management.db`

**解决方案**:
```bash
# Linux/Mac
chmod 664 student_management.db
chown www-data:www-data student_management.db

# Windows
# 右键数据库文件 -> 属性 -> 安全 -> 编辑权限
```

### 3. 静态文件无法访问

**解决方案**:
确保 `static/` 目录存在且包含以下文件：
- login.html
- index.html
- app.js

### 4. 依赖安装失败

**解决方案**:
```bash
# 清理缓存重新安装
pip cache purge
pip install -r requirements.txt --no-cache-dir -i https://pypi.tuna.tsinghua.edu.cn/simple
```

## 🔧 维护指南

### 数据库备份

#### 手动备份
```bash
# 创建备份目录
mkdir -p backups

# 备份数据库
cp student_management.db backups/student_management_$(date +%Y%m%d_%H%M%S).db
```

#### 自动备份（Linux定时任务）
创建备份脚本 `backup.sh`:
```bash
#!/bin/bash
BACKUP_DIR="/opt/backups"
DB_FILE="/opt/student_management_system/student_management.db"
DATE=$(date +%Y%m%d_%H%M%S)

cp $DB_FILE $BACKUP_DIR/student_management_$DATE.db

# 保留最近7天的备份
find $BACKUP_DIR -name "*.db" -mtime +7 -delete
```

添加到crontab:
```bash
crontab -e
# 每天凌晨2点备份
0 2 * * * /opt/student_management_system/backup.sh
```

### 日志管理

#### 配置日志
修改 `main.py`:
```python
import logging

logging.basicConfig(
    level=logging.INFO,
    format='%(asctime)s - %(name)s - %(levelname)s - %(message)s',
    handlers=[
        logging.FileHandler('student_management.log'),
        logging.StreamHandler()
    ]
)
```

#### 日志轮转
创建 `/etc/logrotate.d/student-management`:
```
/opt/student_management_system/*.log {
    daily
    rotate 7
    compress
    missingok
    notifempty
}
```

### 性能优化

#### 1. 启用Gzip压缩
在Nginx配置中添加：
```nginx
gzip on;
gzip_types text/plain text/css application/json application/javascript;
```

#### 2. 数据库优化
定期执行SQLite优化：
```bash
sqlite3 student_management.db "VACUUM;"
sqlite3 student_management.db "ANALYZE;"
```

#### 3. 增加worker数量
Gunicorn启动时调整worker数量：
```bash
gunicorn -w 8 -k uvicorn.workers.UvicornWorker -b 0.0.0.0:8000 main:app
```

### 安全加固

#### 1. 修改默认密码
登录后立即修改admin密码

#### 2. 配置防火墙
```bash
# Linux防火墙配置
sudo ufw allow 8000/tcp
sudo ufw enable
```

#### 3. 启用HTTPS
使用Let's Encrypt免费证书：
```bash
sudo apt install certbot python3-certbot-nginx
sudo certbot --nginx -d your-domain.com
```
